## Quick Start

```bash
# Clone the repository
git clone https://github.com/ariffazil/career-timeline.git
cd career-timeline

# Open locally (no build required)
open index.html          # macOS
start index.html         # Windows
xdg-open index.html      # Linux
```

Or deploy directly to GitHub Pages — the `index.html` is ready to serve.
